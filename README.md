# Guesthouse Management System (MERN)

## Overview

This project is a full-stack Guesthouse Management System developed to manage accommodation requests, bookings, and administrative operations.

The system provides a user-friendly booking interface for applicants and a comprehensive admin dashboard for managing bookings, rooms, and invoices.

---

## Features

### User Features

* Multi-step booking form with validation
* Guest management (multiple guests per booking)
* Booking status tracking
* File upload for ID verification
* Responsive UI for seamless user experience

### Admin Features

* Admin login with protected routes
* Dashboard for managing bookings
* Room allocation and availability tracking
* Booking calendar view
* Invoice management system
